民泊yamaai-山合- は
福岡県糸島市の山間、小さな集落の中にあります。
田園風景の広がるこの集落には、
昔ながらの生活が垣間見えます。
刻々と変わる、山の空気、山の生きものたちに
囲まれた営みの中で
それぞれの心地のいい時間をお過ごしください。
